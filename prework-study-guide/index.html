Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element as well as keyword elements in metadata are critical to SEO.
  • Always ensure headers H1,H2 etc. are used in the correct order. Only use H1 once per page.
  • Nested elements are called child elements, the element that holds them is the parent.
  • Use Semantic HTML elements whenever possible, creates the most accesible page. Non-semantic elements can still be the most efficient option in certain cases

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors in CSS style sheets can be defined by name or by Hexadecimal codes
  • CSS style sheets must be linked properly to the main code file for the style to take effect
  • Class attributes can be designated and specific styles assigned to them, rather than by element
